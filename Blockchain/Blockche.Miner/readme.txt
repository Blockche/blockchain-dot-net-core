NoChain CPU Miner

Start
Download and run the exe file.

Arguments
--threads - number of threads (default is 1)
--url - url of the node or pool (default is http://nochainpool.azurewebsites.net)
--address - address of the miner when not mining on pool (default is 1234567890123456789012345678901234567890)
--user - user when mining in pool (default is Hardcore)
--worker - worker when mining in pool (default is Slave1)
--pool - flag if mining should mine in pool
--test - run in benchmark mode
